# 双色球自适应预测引擎

这个目录提供一个可复跑的双色球历史回测与预算组合脚本，核心流程是：

1. 读取开奖 Excel，把每 7 行整理成 1 期开奖记录。
2. 从 `2019069 -> 2020017` 开始建立首个训练窗口。
3. 对每个后续期号执行“训练 -> 预测下一期 -> 对照真实开奖 -> 在线更新权重”。
4. 在 `10 / 50 / 100 / 200 / 500 / 1000` 元预算下，自动选择更适合当前预算的投注结构。

当前版本相对初版新增了三层自适应能力：

- 红球和蓝球分开做在线调权
- 启发式策略层和在线二分类模型层做二层融合
- 预算结构选择同时参考历史 EMA、当前方案代理分数和探索项
- 引入相似期检索特征和基于真实奖级回收的回报校准器
- 自动接入中国福彩网官方开奖接口，补充销售额、奖池、一二等奖注数/金额和一等奖地区分布
- 预算选择目标已从“高 ROI 小成本”修正为“预算内更高回报”，并加入预算利用率约束

已实现的投注结构：

- 单式组合
- 红球复式
- 蓝球复式
- 全复式
- 胆拖

运行方式：

```bash
python run.py \
  --excel /path/to/双色球最近1000期开奖结果.xlsx \
  --rule-doc /path/to/中国福利彩票双色球游戏规则.docx \
  --output-dir /path/to/outputs
```

> 将 `/path/to/` 替换为实际文件路径。

离线模式（跳过官方接口、使用本地缓存）：

```bash
python run.py \
  --excel /path/to/双色球最近1000期开奖结果.xlsx \
  --rule-doc /path/to/中国福利彩票双色球游戏规则.docx \
  --output-dir /path/to/outputs \
  --no-official
```

主要输出：

- `budget_summary.csv`：各预算回测汇总
- `rolling_backtest.csv`：每一期滚动预测的回测明细
- `prediction_next_issue.json`：下一期详细投注方案
- `prediction_report.md`：人类可读报告
- `adaptive_diagnostics.json`：自适应权重、模型系数和策略质量诊断
- `official_enrichment.json`：官方数据同步与覆盖情况
- `official_draws.json`：中国福彩网接口抓取缓存

注意：

- 中国福彩网接口可提供当期一等奖、二等奖的单注奖金，因此在官方数据覆盖的期次上，脚本会对一至六等奖做精确金额回测。
- 如果网络不可用，脚本会优先回退到输出目录中的 `official_draws.json` 缓存；缓存也没有时才只使用 Excel。
- 历史开奖无法保证未来开奖可预测，这个引擎更适合作为历史拟合和预算覆盖分析工具。
